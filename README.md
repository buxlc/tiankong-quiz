# 📝 填空题题库 - 智能刷题工具

一款纯前端的填空题自动识别与刷题工具。上传 Word 文档（.docx），自动识别下划线内容生成填空题，支持顺序/随机出题、自动批改、复习模式等功能。

**👉 在线使用：[https://buxlc.github.io/tiankong-quiz/](https://buxlc.github.io/tiankong-quiz/)**

## ✨ 功能特性

- **智能解析** — 上传 `.docx` 文件，自动识别下划线格式文本为填空答案，生成填空题
- **批量导入** — 支持拖拽或选择多个文件，一次导入整个题库
- **本地持久化** — 题目自动保存在浏览器 IndexedDB 中，关闭再打开题库仍在，多次上传自动追加
- **顺序/随机出题** — 🎲 随机模式或 📑 顺序模式，一键切换
- **灵活选题量** — 预设 1/3/5/7/10/全部，支持自定义任意数量
- **上一批/下一批** — 支持来回切换已做和未做的题目批次
- **自动批改** — 标注正确(绿)/错误(红)，显示正确答案和正确率
- **答案宽松匹配** — 忽略所有标点符号和空格，中英文标点、大小写均兼容
- **复习模式** — 一键开关，原文以半透明样式覆盖在填空位置，方便对照记忆
- **已批改历史** — 批改记录按批次保存，可选择查看任意一批的批改结果
- **可拖动复习按钮** — 右下角悬浮按钮可自由拖动，手机/电脑均适配
- **移动端适配** — 响应式设计，手机端完美使用
- **清空题库** — 一键清空重新开始，适合换题库使用

## 📦 使用方法

### 在线使用（推荐）
直接访问 [https://buxlc.github.io/tiankong-quiz/](https://buxlc.github.io/tiankong-quiz/)

### 本地使用
下载 `index.html` 文件，用浏览器直接打开即可，无需安装任何环境。

## 📄 Word 文档格式要求

在 Word 中对答案部分添加**下划线格式**，工具会自动将下划线内容替换为填空位。

示例：
```
燕山构造阶段属于<u>中生代</u>时期。

寒武纪最重要的标准化石是<u>三叶虫</u>。
```
其中**下划线部分**会被识别为答案，生成填空题时该位置会显示为空白输入框。

## 🛠 技术栈

- **纯前端** — 无需后端服务器，单文件即可运行
- **原生 ZIP 解析** — 基于浏览器 `DecompressionStream` API 解压 `.docx`
- **XML 解析** — 解析 Word 文档的 `document.xml` 提取下划线文本
- **IndexedDB** — 本地持久化存储，无需网络
- **GitHub Pages** — 静态托管，零成本部署
